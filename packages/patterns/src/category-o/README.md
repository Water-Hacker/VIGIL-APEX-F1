# Category O — Extractive Sector (EITI + NRGI typologies)

**Source bodies:** Extractive Industries Transparency Initiative
(EITI Standard 2.5, 2.6); Natural Resource Governance Institute
(NRGI) Resource Governance Index; OCCRP investigations into African
mining/oil concessions.

**Why this category exists:** Cameroon's extractives sector
(SONARA refinery, oil blocks in the Rio del Rey + Douala/Kribi
basins, iron ore in Mbalam, bauxite in Minim-Martap) is a known
high-risk corruption surface. The 2010 SONARA fire investigation
and subsequent litigation surfaced specific patterns.

**Patterns (3):**

| ID      | EN title                                                                              | Source                                        |
| ------- | ------------------------------------------------------------------------------------- | --------------------------------------------- |
| P-O-001 | Mining concession awarded without environmental impact assessment                     | EITI 2.5 + Cameroon Loi 2016/017 art. 24      |
| P-O-002 | Oil / gas block transferred to politically-connected entity at below-market valuation | NRGI Resource Governance Index methodology    |
| P-O-003 | Production-sharing rate anomalously favourable to operator                            | EITI 2.6 + IMF FAD fiscal-regime benchmarking |
