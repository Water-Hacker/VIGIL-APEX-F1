# Category L — Foreign Bribery & Corruption (OECD typologies)

**Source body:** OECD Working Group on Bribery in International
Business Transactions, published as _OECD Foreign Bribery Report —
An Analysis of the Crime of Bribery of Foreign Public Officials_
(2014, updated 2022). Per Article 1 of the OECD Anti-Bribery Convention.

**Coverage:** the dominant typologies surfaced across 800+ analysed
bribery cases. Of particular relevance: 41 % of cases involved
kickbacks; 71 % used intermediaries; payments concentrated in
extractives + infrastructure + transport sectors.

**Patterns (5):**

| ID      | EN title                                                | OECD typology                                     |
| ------- | ------------------------------------------------------- | ------------------------------------------------- |
| P-L-001 | Third-party agent / intermediary with no substance      | OECD Foreign Bribery Report §3.4 — agents pattern |
| P-L-002 | Agent fee abnormally high vs. industry benchmark        | OECD: inflated commission                         |
| P-L-003 | Offshore consultancy payment concurrent with award      | OECD: payment via offshore third party            |
| P-L-004 | Pre-award gift / travel to decision-maker               | OECD: hospitality conversion                      |
| P-L-005 | Post-award employment of decision-maker / family member | OECD / FCPA: revolving-door pattern               |
