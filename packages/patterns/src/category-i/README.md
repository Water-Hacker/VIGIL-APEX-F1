# Category I — Asset Misappropriation (ACFE Fraud Tree)

**Source body:** Association of Certified Fraud Examiners (ACFE) —
_Report to the Nations_ (latest edition; biennial publication) and
the canonical ACFE Fraud Tree v2.

**Citation:** ACFE classifies occupational fraud into three major
branches: **corruption**, **asset misappropriation**, **fraudulent
financial reporting**. Asset misappropriation is the most common
(~85% of cases) and the smallest median loss (~$120K vs ~$700K for
fraudulent reporting). This category implements the misappropriation
sub-tree.

**Sub-tree mapped to VIGIL patterns (8):**

| ID      | EN title                              | ACFE Fraud Tree node                                                                         |
| ------- | ------------------------------------- | -------------------------------------------------------------------------------------------- |
| P-I-001 | Shell-vendor billing scheme           | Asset Misappropriation → Cash → Disbursements → Billing → Shell Company                      |
| P-I-002 | Personal purchases on state account   | Asset Misappropriation → Cash → Disbursements → Billing → Personal Purchases                 |
| P-I-003 | Ghost employee on payroll             | Asset Misappropriation → Cash → Disbursements → Payroll → Ghost Employees                    |
| P-I-004 | Falsified wages / overtime            | Asset Misappropriation → Cash → Disbursements → Payroll → Falsified Wages                    |
| P-I-005 | Fictitious expense reimbursement      | Asset Misappropriation → Cash → Disbursements → Expense Reimbursements → Fictitious Expenses |
| P-I-006 | Cash skimming (unrecorded receipts)   | Asset Misappropriation → Cash → Theft of Cash on Hand → Skimming                             |
| P-I-007 | Cash larceny (recorded then stolen)   | Asset Misappropriation → Cash → Theft of Cash on Hand → Larceny                              |
| P-I-008 | Inventory / non-cash misappropriation | Asset Misappropriation → Inventory / Other Assets                                            |

**Why this matters for Cameroon:** ACFE's African chapter regional
reports flag that public-sector asset misappropriation accounts for a
disproportionate share of African fraud cases (~38% of African cases
vs ~24% globally per the 2024 Report to the Nations). The original
43 Cameroon-tuned patterns A–H cover procurement-side fraud well but
miss the asset-side branch entirely. This category closes that gap.
