# Category K — Trade-Based Money Laundering (FATF Typologies)

**Source body:** Financial Action Task Force (FATF) — _Trade-Based
Money Laundering: Trends and Developments_ (2020) and predecessor
guidance from 2006 / 2012. Co-published with the Egmont Group of FIUs.

**Coverage:** the six FATF-named TBML typologies. State-procurement
fraud frequently launders through trade misinvoicing, especially in
extractives-rich African economies (FATF Mutual Evaluation Reports
for CEMAC region cite TBML as a key vulnerability).

**Patterns (6):**

| ID      | EN title                | FATF typology                                     |
| ------- | ----------------------- | ------------------------------------------------- |
| P-K-001 | Over-invoicing          | Trade-Based ML: over-invoicing of imports/exports |
| P-K-002 | Under-invoicing         | Trade-Based ML: under-invoicing                   |
| P-K-003 | Multiple invoicing      | Trade-Based ML: multiple invoicing                |
| P-K-004 | Phantom shipment        | Trade-Based ML: invoicing without delivery        |
| P-K-005 | Goods misclassification | Trade-Based ML: HS code misclassification         |
| P-K-006 | Round-trip trade        | Trade-Based ML: round-tripping                    |

References: FATF Recommendation 32 (cash couriers + trade-based ML
disclosure), Wolfsberg correspondent-banking principles, BEAC payment-
system anomaly indicators.
