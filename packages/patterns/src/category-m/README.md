# Category M — Procurement Collusion (World Bank Integrity Vice Presidency)

**Source body:** World Bank INT (Integrity Vice Presidency) —
debarment policy + _Guide to the Disclosure of Information Related to
Sanctions Cases_; OECD _Guidelines for Fighting Bid Rigging_.

**Patterns (4)** — the four classical bid-rigging schemes:

| ID      | EN title              | World Bank INT / OECD typology               |
| ------- | --------------------- | -------------------------------------------- |
| P-M-001 | Bid rotation          | Suppliers take turns winning                 |
| P-M-002 | Bid suppression       | Bidders deliberately withdraw to clear win   |
| P-M-003 | Complementary bidding | Losing bids deliberately uncompetitive       |
| P-M-004 | Market allocation     | Geographic / category carve-up among bidders |
