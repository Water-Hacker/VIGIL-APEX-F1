# Category J — Financial Statement Fraud (ACFE Fraud Tree)

**Source body:** Association of Certified Fraud Examiners (ACFE) —
_Report to the Nations_ and ACFE Fraud Tree v2.

**Coverage:** financial-statement-fraud branch of the ACFE tree.
Median loss is the largest of all branches (~$700K) though
frequency is the lowest (~5–9% of occupational fraud). Highly
relevant where state contractors publish audited statements as part
of procurement bid documentation.

**Patterns (5):**

| ID      | EN title                              | ACFE node                                                           |
| ------- | ------------------------------------- | ------------------------------------------------------------------- |
| P-J-001 | Premature / fictitious revenue        | Financial Statement Fraud → Revenue Recognition → Premature         |
| P-J-002 | Overstated asset valuation            | Financial Statement Fraud → Asset Valuation → Overstatement         |
| P-J-003 | Concealed off-books liabilities       | Financial Statement Fraud → Liabilities / Expenses → Concealment    |
| P-J-004 | Expense capitalisation (deferral)     | Financial Statement Fraud → Liabilities / Expenses → Capitalisation |
| P-J-005 | Undisclosed related-party transaction | Financial Statement Fraud → Improper Disclosures → Related-Party    |
