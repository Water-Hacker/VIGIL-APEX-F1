# Category P — Post-Award Personal Enrichment

**Source bodies:** OECD Foreign Bribery Report (revolving door, lifestyle indicators);
FATF Recommendation 32 (PEP / family transactions); OCCRP investigations
(post-award asset acquisitions by officials).

**Why this category exists:** Cameroon's _Patrimoine Inexpliqué_
(unexplained wealth) doctrine — Loi 2018/011 — empowers ANIF to
investigate sudden asset acquisitions by public officials. These
patterns mechanise the detection that ANIF currently does manually
from CONAC referrals.

**Patterns (3):**

| ID      | EN title                                                                             | Source                        |
| ------- | ------------------------------------------------------------------------------------ | ----------------------------- |
| P-P-001 | Property flip — official purchases post-award at suspicious-low price                | OECD / OCCRP                  |
| P-P-002 | Land title transferred to family member of state official within 90 days of contract | FATF R.32 / ANIF Loi 2018/011 |
| P-P-003 | Real estate purchase by contractor's UBO concurrent with state payment               | OCCRP / Pandora Papers        |
