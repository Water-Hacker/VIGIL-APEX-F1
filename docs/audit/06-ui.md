## Lighthouse / axe-core run plan (deferred)

When the architect runs the dashboard locally:

```bash
# Start dev server
pnpm --filter @vigil/dashboard dev

# In a second shell, install lighthouse if needed
npm install -g lighthouse chrome-launcher

# Run against each high-traffic screen
for url in \
  http://localhost:3000/findings \
  http://localhost:3000/findings/[id] \
  http://localhost:3000/council/proposals \
  http://localhost:3000/tip \
  http://localhost:3000/verify \
  http://localhost:3000/audit/ai-safety; do
  name=$(echo "$url" | sed 's|.*/||; s/[^a-z0-9]/_/g')
  lighthouse "$url" --output=json --output=html \
    --output-path=docs/audit/evidence/lighthouse/$name.html \
    --only-categories=accessibility,performance,best-practices
done

# Aggregate axe-core run via @axe-core/cli
npx @axe-core/cli http://localhost:3000/findings --save docs/audit/evidence/lighthouse/axe-findings.json
```

Expected: every page ≥ 95 on accessibility (per audit spec).
Save reports to `docs/audit/evidence/lighthouse/`.
