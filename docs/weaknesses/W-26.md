# W-26 — No formal Phase-0 dry-run sign-off

| Field | Value |
|---|---|
| ID | W-26 |
| Severity | **Medium** |
| Status | **fix committed** |
| Authoritative ref (in TRUTH or repo) | docs/decisions/DRY-RUN-DECISION.md |
| Source(s) in original docs | EXEC §26-30 |

## Problem

EXEC §26-30 specifies the dry-run protocol with GO / GO-with-note / PATCH / REWORK outcomes. There is no decision-record artefact whose existence gates Phase 0 commit-on-main.

## Fix

**`docs/decisions/DRY-RUN-DECISION.md` template** (committed); CI lint refuses to merge any Phase-0 commit to `main` unless this file exists with `Status: GO` or `GO-with-note`. Status `PATCH` or `REWORK` triggers documentation revision cycle and a new dry-run.
