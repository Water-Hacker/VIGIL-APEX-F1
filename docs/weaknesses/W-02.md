# W-02 — NAS model & RAID drift

| Field | Value |
|---|---|
| ID | W-02 |
| Severity | **Medium** |
| Status | **fix committed** |
| Authoritative ref (in TRUTH or repo) | TRUTH.md Section B |
| Source(s) in original docs | SRD §02.2, MVP §02.2 |

## Problem

SRD §02.2 specifies Synology RS3621XS+ (rackmount, 24 TB usable, RAID-6). MVP §02.2 specifies Synology DS1823xs+ pair (desktop, 32 TB usable per unit, SHR-2, geographically separated). The pair model is *better* than the rackmount because of geographic separation, but the SRD is unpatched.

## Fix

**Adopt MVP §02.2 (DS1823xs+ pair) as canonical.** SRD §02.2 must be patched in the next amendment cycle. The pair model is committed in TRUTH.md Section B.
