## Problem

SRD §10.5 has manual fallback to Wayback Machine; BUILD-V2 §43-44 has the first-contact protocol that archives broken HTML. There is no LLM-assisted selector regeneration loop.

## Fix

**Add `worker-adapter-repair`**: when an adapter fails to parse for >24 h:

1. Freeze the live HTML at `infra/sites/<id>/<timestamp>.html`.
2. Ask Claude Sonnet 4.6 to propose new selectors using the spec'd schema as ground truth (anti-hallucination meta-wrapper applies; cite source HTML char spans).
3. Run the proposed selectors against the same HTML in a sandbox.
4. Synthetic-test pass gates promotion via a GitHub Actions PR that the architect reviews.
5. PR merged → adapter reloaded; PR rejected → first-contact protocol still applies (operator alerted).

This ships in Phase 2 (MOU-gated escalation phase) since it materially reduces architect maintenance overhead.
