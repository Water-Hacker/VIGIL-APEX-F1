# W-25 — CONAC delivery format asserted not negotiated

| Field | Value |
|---|---|
| ID | W-25 |
| Severity | **High** |
| Status | **fix proposed** |
| Authoritative ref (in TRUTH or repo) | TRUTH.md Section G |
| Source(s) in original docs | SRD §25, EXEC §15 |

## Problem

SRD §25 specifies SFTP + manifest schema as if CONAC has agreed. EXEC §15 admits this is unproven (60-90 day non-objection clause). No Plan B if CONAC requires a totally different format.

## Fix

**Build CONAC SFTP module behind a format-adapter layer**. The manifest schema is plug-in; `worker-conac-sftp` reads which schema version + delivery method to use from `docs/decisions/log.md` decision pointer. Schema change after engagement-letter response = config update + new format-adapter module, no rewrite.

**Plan B target: Cour des Comptes** (whose ingestion preferences are documented in their public correspondence guide). Dossier-routing decision is itself a council vote per SRD §23.
