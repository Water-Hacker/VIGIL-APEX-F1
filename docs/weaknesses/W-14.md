## Problem

SRD §20 enumerates twelve layers of anti-hallucination control but does not show their unit-test contracts. Each control is conceptual.

## Fix

**Write `prompts/synthetic-hallucinations.jsonl`** — 200 known-bad LLM outputs covering each layer's failure modes (fabricated citations, miscounted amounts, invented entities, schema violations, language drift, etc.). CI nightly + on PR (touching `packages/llm/`) asserts each layer rejects ≥ 95% of its corresponding category.

Corpus authored in M0c week 1, expanded with each new pattern, audit-trailed in `packages/llm-prompts/__tests__/`.
