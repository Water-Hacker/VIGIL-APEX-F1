# W-08 — OpenPGP signing key has no break-glass

| Field | Value |
|---|---|
| ID | W-08 |
| Severity | **High** |
| Status | **fix proposed** |
| Authoritative ref (in TRUTH or repo) | TRUTH.md Section E |
| Source(s) in original docs | HSK §4.5, HSK §8.5 |

## Problem

HSK §4.5 admits dual-key loss = permanent loss of the CONAC signing identity. The §8.5 'both keys lost simultaneously' protocol explicitly says: 'OpenPGP signing key: this is the only credential with no recovery path by design.' This is a single point of irreversible cryptographic failure for institutional dossier delivery.

## Fix

**Add a 9th 'deep-cold' YubiKey** holding the same OpenPGP master key, sealed in an off-jurisdiction bank safe-deposit box (Geneva, Lisbon, or Zurich — to be decided in the decision log). HSK already permits one-time secret-key transfer between cards under controlled conditions; do this once at provisioning, with a notarial witness.

Also: pre-publish a **dossier-signing-key-rotation protocol** with CONAC up-front, so a future rotation is institutional, not improvised. Total YubiKey estate becomes 9.
